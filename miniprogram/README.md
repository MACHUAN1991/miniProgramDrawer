# 知识抽屉 - 微信小程序

帮助你在兴趣来临时快速收藏知识，定期回顾不忘记忆。

## 功能

- **快速收藏**：支持文字和图片收藏
- **分类筛选**：按全部/文字/图片类型筛选
- **每日推荐**：每天自动推荐一条旧收藏
- **触底加载**：支持上拉加载更多
- **返回顶部**：快速回到页面顶部
- **搜索功能**：支持关键词搜索已收藏内容
- **长文展开**：超过100字的内容可展开/收起
- **云端同步**：换手机也不丢数据
- **用户系统**：支持设置头像和昵称

## 使用步骤

### 1. 开通云开发

1. 在微信公众平台后台 → 开发 → 开发管理 → 开发设置 中，开通云开发
2. 获取云环境ID，替换 `app.js` 中的 `env` 值

### 2. 创建数据库集合

在云开发控制台创建以下集合：
- `clips` - 收藏记录

### 3. 配置小程序

确保在微信公众平台开启了：
- 基础库版本
- 云开发能力

## 项目结构

```
miniprogram/
├── app.js              # 小程序入口
├── app.json            # 小程序配置
├── app.wxss            # 全局样式
├── assets/             # 图标资源
├── cloud/
│   ├── deleteClip/     # 删除内容云函数
│   └── uploadImage/    # 图片上传云函数
└── pages/
    └── index/          # 首页 - 收藏列表
        ├── index.js    # 页面逻辑
        ├── index.wxml  # 页面结构
        ├── index.wxss  # 页面样式
        └── index.json  # 页面配置
```

## 数据结构

### clips 集合

| 字段 | 类型 | 说明 |
|------|------|------|
| type | string | 'text' 或 'image' |
| content | string | 文字内容 |
| imageUrl | string | 云存储图片ID |
| creatorName | string | 创建者昵称 |
| creatorAvatar | string | 创建者头像 |
| createdAt | string | 创建时间 (ISO格式) |

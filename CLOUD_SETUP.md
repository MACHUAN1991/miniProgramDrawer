# 云开发配置指南

## 第一步：开通云开发

### 1. 登录微信公众平台
访问：https://mp.weixin.qq.com

登录你的小程序账号

### 2. 开通云开发
1. 进入 **开发 → 开发管理 → 开发设置**
2. 找到 **云开发** 模块
3. 点击 **开通**
4. 阅读协议并确认

### 3. 创建云环境
1. 开通后会自动跳转到云开发控制台
2. 点击 **环境列表 → 新建环境**
3. 环境名称：填写 `knowledge-drawer` 或其他名称
4. 环境ID：系统自动生成，格式如 `cloud1-xxxxx`，**复制这个ID**

---

## 第二步：配置小程序

### 1. 复制云环境ID
在 `app.js` 中替换 `env` 值：

```javascript
wx.cloud.init({
  env: '你的云环境ID',  // 例如：cloud1-8a3b2c1d
  traceUser: true,
});
```

---

## 第三步：创建数据库集合

### 1. 进入云开发控制台
在微信开发者工具中：
- 点击顶部 **云开发** 按钮
- 或访问 https://cloud.weixin.qq.com

### 2. 创建集合

点击 **数据库 → 创建集合**，按顺序创建以下3个集合：

#### clips（收藏记录）
- 权限设置：**自定义安全规则**
- 安全规则：
```json
{
  "read": true,
  "write": true
}
```

#### topics（主题）
- 权限设置：与上面相同

#### users（用户设置）
- 权限设置：与上面相同

---

## 第四步：部署云函数

### 1. 安装依赖
在微信开发者工具中：
1. 打开 `cloud/ocr` 文件夹
2. 右键 `package.json`
3. 选择 **安装依赖**

### 2. 部署云函数
1. 右键 `cloud/ocr` 文件夹
2. 选择 **上传并部署**
3. 确认部署到你的云环境

---

## 第五步：添加图标资源

在 `miniprogram/assets/` 文件夹中添加以下图标（PNG格式，64x64像素）：

| 文件名 | 用途 |
|--------|------|
| home.png | 首页图标 |
| home-active.png | 首页选中图标 |
| add.png | 收藏图标 |
| add-active.png | 收藏选中图标 |
| mine.png | 我的图标 |
| mine-active.png | 我的选中图标 |
| camera.png | 拍照按钮 |
| album.png | 相册按钮 |
| review.png | 回顾图标 |
| notify.png | 通知图标 |
| about.png | 关于图标 |
| empty.png | 空状态图标 |

**提示**：可以使用阿里巴巴图标库（iconfont.cn）免费下载

---

## 第六步：配置订阅消息

### 1. 添加模板
在微信公众平台：
1. 进入 **功能 → 订阅消息**
2. 点击 **添加模板**
3. 搜索并添加：`每日提醒` 或类似模板

### 2. 获取模板ID
记录下模板ID，用于后续推送功能

---

## 验证配置

完成以上步骤后：

1. 在微信开发者工具中点击 **编译**
2. 点击底部 **云开发** 图标查看是否连接成功
3. 尝试新建一个主题，测试数据是否保存

如果遇到问题，检查：
- 云环境ID是否正确
- 云函数是否部署成功
- 数据库集合是否创建
